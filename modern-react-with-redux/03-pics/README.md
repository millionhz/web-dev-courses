# Events and APIs

The App uses:

- react-scripts
- [Semantic UI](https://semantic-ui.com/)
- Unsplash API
- axios

## API keys

The app requires a file named `keys.js` on the `src` folder containing unsplash API credentials.

`keys.js` template:
```js
export const access = 'access-key-here';
export const secret = 'secret-key-here';
```
